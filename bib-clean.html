Cleaning BibDesk Bib Files with Python

(Last updated: September 2, 2025)

I use BibDesk to manage my bibliography. When exporting, BibDesk often adds many non-standard fields such as date-added, date-modified, priority, file, and bdsk-url-1. These fields are useless for LaTeX. I prefer to remove them while keeping all academic fields (author, title, journal, year, doi, url, etc.).

Workflow (MacOS)

1. Manage references in BibDesk.
2. Export the database with all fields.
3. Download clean_bib.py 4. Put the Python script and your .bib file in the same folder, and run the script with terminal (make sure to locate yourself in the targeted folder first): python3 clean_bib.py input.bib output.bib
